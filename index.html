Descripción del proyecto

Este proyecto presenta el diseño, programación y simulación de un sistema basado en controladores lógicos programables (PLC) para automatizar un proceso industrial. Se emplearon herramientas de software y hardware para garantizar un funcionamiento eficiente y seguro.

Resultados obtenidos

Durante la simulación se observó un comportamiento estable en las entradas y salidas del PLC, cumpliendo con los parámetros establecidos. El sistema responde adecuadamente a los eventos de control, mostrando la viabilidad del diseño.
